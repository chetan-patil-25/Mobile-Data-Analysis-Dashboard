# Mobile Data Analysis Dashboard

## Overview
This project presents an interactive Power BI dashboard for analyzing mobile phone market data. The dashboard provides insights into mobile brands, pricing trends, RAM storage, operating systems, and USB types through dynamic visualizations and KPI metrics.

## Features
- Brand-wise mobile distribution analysis
- Average price comparison across brands
- Operating system market share visualization
- RAM storage vs price analysis
- USB type distribution insights
- Interactive filters and slicers

## Dashboard KPIs
- Average Mobile Price
- Total Mobile Count
- Brand Performance Analysis
- Operating System Distribution

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Skills Demonstrated
- Data Cleaning

## Key Insights
- Apple has the highest average mobile price.
- Samsung leads in total mobile availability.
- Android and iOS dominate the operating system market.
- 4 GB and 8 GB RAM devices account for the highest market value.
- Type-C is the most commonly used USB type.
